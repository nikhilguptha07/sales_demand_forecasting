# Sales Demand Forecasting Pipeline
An end-to-end machine learning and data analytics project focused on predicting superstore sales demand using advanced regression techniques and visualizing key business insights.
## 📊 Project Overview
* **Machine Learning:** Built a predictive model using **XGBoost Regressor** to forecast future sales demand based on historical data.
* **Data Visualization:** Designed an interactive **Power BI Dashboard** (`salesdemandforecast3.pbix`) to track actual vs. forecasted sales performance, trends, and product categories.
* **Datasets Used:** Includes the historical sales records (`Sample - Superstore.csv`) and the generated predictions (`superstore_forecast.csv`).
## 🛠️ Tech Stack
* **Language:** Python (Jupyter Notebook)
* **Libraries:** XGBoost, Pandas, NumPy, Scikit-learn
* **BI Tool:** Power BI Desktop
* **Version Control:** Git & GitHub
